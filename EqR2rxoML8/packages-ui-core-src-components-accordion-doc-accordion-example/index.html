Списание абонентской платы

При подключении тарифа на вашем счёте должна быть сумма, достаточная для списания абонентской платы за месяц и дополнительных услуг, указанных в заявке. В день перехода на тариф списывается полная сумма абонентской платы, далее списания происходят раз в месяц в тот же день.

Подключение к интернету

Подключение к домашнему интернету осуществляется в удобное для вас время по технологиям Ethernet, Docsis. Монтажник проведёт все необходимые работы под ключ.
